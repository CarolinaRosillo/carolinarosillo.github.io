Pruebas automatizadas para Urban Routes

Probé la funcionalidad de la aplicación durante todo el proceso de pedido de taxi, asegurando la correcta configuración de rutas, selección de tarifas, verificación de números de teléfono y gestión de métodos de pago. Implementé el patrón de diseño Page Object Model (POM) para mantener un código modular y mantenible, utilizando Selenium WebDriver y Pytest en PyCharm. Realicé pruebas de regresión para verificar la estabilidad de la aplicación tras actualizaciones. Este trabajo ayudó a identificar errores críticos, mejorando la estabilidad y la experiencia del usuario final.

Pruebas de API Automatizadas para Urban Grocers

Realicé pruebas de API utilizando Postman para validar la incorporación de productos y la autenticación de usuarios. Gestioné el repositorio en GitHub y utilicé la línea de comandos para acceder a registros en un servidor remoto y optimizar consultas SQL. Implementé pruebas automatizadas con Python, Pytest y Requests, abarcando tanto pruebas unitarias para asegurar la funcionalidad de componentes individuales como pruebas de integración para verificar la correcta interacción entre servicios. Estas prácticas mejoraron la cobertura de pruebas y permitieron una detección más temprana de errores, garantizando una mayor calidad en el sistema.
